Codelab Exercise 1

Web Research with Antigravity

This exercise demonstrates how Google Antigravity can autonomously browse the web, extract information, and present structured results — all from a simple prompt.

How It Works

1

Give a Prompt

Tell the agent what site to visit and what data to extract.

2

Agent Browses

Antigravity opens a browser, navigates, and reads the page content.

3

Data Extraction

The agent identifies relevant information and structures it.

4

Results Artifact

Clean, formatted results are returned as an artifact you can review.

Prompts to Try in Antigravity

Basic

Get today's top news headlines from Google News

Visita news.google.com y muéstrame los momentos destacados del momento

The agent will open Google News, scan the headlines, and return a summary of the current top stories organized by topic, including sources and publication times.

Advanced

Get Google Cloud release notes categorized by product

Visit https://cloud.google.com/release-notes and get me a summary of the release notes, categorized by product.

This demonstrates deeper data extraction — the agent must navigate a complex documentation site, identify product categories, and group release notes for a structured summary.

Try Your Own

Visit any website with dashboards or data tables

Visit [YOUR_URL_HERE] and extract the key metrics / data from the dashboard. Summarize the findings.

Replace the URL with any publicly accessible site that has charts, tables, or structured data. The agent will attempt to read and summarize the visual and textual content.

What to Expect

Browser Control

🧠

Thought Process

📄
