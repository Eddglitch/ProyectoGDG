Exercise 01

Web
Research

Antigravity autonomously browses the web, extracts information, and presents structured results — all from a simple prompt.

🌐
agent:browsing_

How It Works

// 04 steps
01

Give a Prompt

Tell the agent what site to visit and what data to extract.

02

Agent Browses

Antigravity opens a browser, navigates, and reads the content.

03

Data Extraction

The agent identifies relevant information and structures it.

04

Results Artifact

Clean, formatted results returned as an artifact you can review.

Try These Prompts

// 03 examples
Basic

Get today's top news headlines from Google News

Visita news.google.com y muéstrame los momentos destacados del momento

The agent will scan headlines and return a summary organized by topic.

Advanced

Get Google Cloud release notes categorized by product

Visit https://cloud.google.com/release-notes and get me a summary categorized by product.

Deeper data extraction — the agent navigates complex documentation sites.

Custom

Visit any website with dashboards or data tables

Visit [YOUR_URL] and extract the key metrics from the dashboard. Summarize findings.

Replace the URL with any publicly accessible site with structured data.
